View: Skills Menu

Overview

Property Value
Route /skills (slash command)
Auth Required N/A (local CLI)
Layout Split-panel (categories/skills)

What Does This View Do?

The Skills Menu provides an interactive terminal UI for browsing, installing, and managing agent skills. Skills are reusable code patterns and workflows that extend the agent's capabilities. Users can browse a remote catalog, install skills locally, and manage their skill directories.

Command Line Interface

Edge Cases & Gotchas

WARNING

Remote catalog unavailable: If the remote catalog cannot be loaded, the menu shows an empty state. Skills can still be added manually.

NOTE

Skill validation: Installed skills are validated to ensure they have a valid SKILL.md file before being recognized.

TIP

Manual installation: Skills can be manually placed in ~/.code_puppy/skills/ and then discovered with /skills refresh.

CAUTION

Skill conflicts: Installing a skill with the same ID as an existing one will overwrite it.
