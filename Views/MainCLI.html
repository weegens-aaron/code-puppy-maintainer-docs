View: Main CLI

Overview

Property Value
Entry Point code-puppy or python -m code_puppy
Mode Interactive Terminal UI
Input Method prompt_toolkit with enhanced completions

What Does This View Do?

The Main CLI is the primary user interface for Code Puppy. It provides an interactive terminal-based environment where users can:

  • Submit natural language coding tasks to AI agents
  • Execute shell commands directly
  • Manage agents, models, and configuration
  • Interact with installed skills and MCP servers
  • View and manage conversation history

User Actions

Action Trigger Result
Submit task Type prompt + Enter Agent processes and responds
Cancel agent Ctrl+C (or configured key) Stops current agent execution
Cancel shell Ctrl+X Interrupts running shell command
Exit Ctrl+D or /exit Graceful shutdown
Path completion Type @ + Tab File path suggestions
Model selection /model Opens model picker menu
Help /help Shows available commands
Shell passthrough !<command> Executes shell command directly
Multiline input Alt+M or F2 Toggle multiline editing mode
Newline in multiline Ctrl+J Insert newline without submitting
Paste image Ctrl+V, F3, or /paste Attach image to prompt

Command Line Arguments

Argument Short Description
--version -v Show version and exit
--interactive -i Run in interactive mode
--prompt -p Execute single prompt and exit
--agent -a Specify agent to use
--model -m Specify model to use
command - Run single command (deprecated)

Examples

Basic Interactive Mode

$ code-puppy
# Shows logo, help messages, then prompt
[code-puppy @ gpt-5] >>> Enter your coding task:

Single Prompt Mode

$ code-puppy -p "Create a Python function to calculate fibonacci"
# Executes and exits after response

With Specific Agent and Model

$ code-puppy --agent code-puppy --model gpt-5
# Uses specified agent and model

Edge Cases & Gotchas

WARNING

Windows + uvx: On Windows when launched via uvx, Ctrl+C is disabled to prevent terminal bricking. Use Ctrl+K instead for cancellation.

NOTE

First-time setup: On first run, the onboarding wizard may automatically launch to guide initial configuration.

TIP

Path completion: Use @ to trigger file path completion. The completion system understands relative and absolute paths.

CAUTION

Shell commands: Commands starting with ! bypass the agent and execute directly. Be careful with destructive commands.
