View: Diff Menu

Overview

Property Value
Route /diff (slash command)
Auth Required N/A (local CLI)
Layout Split-panel (menu/preview)

What Does This View Do?

The Diff Menu provides an interactive terminal UI for configuring diff display colors and previewing diff formats. Users can customize the colors used for additions and deletions in diff output, and preview how diffs will look for various programming languages.

Edge Cases & Gotchas

WARNING

Color visibility: Some color combinations may be hard to read on certain terminal backgrounds.

NOTE

Live preview: Changes are reflected immediately in the preview panel.

TIP

Language cycling: Use ←/→ to quickly preview how diffs look in different languages.

CAUTION

Unsaved changes: If you exit without saving, color changes are lost.
