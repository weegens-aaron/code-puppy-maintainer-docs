What Does This View Do?

The Terminal Web Interface provides a browser-based interactive terminal for Code Puppy. It uses WebSockets to connect to a PTY (pseudo-terminal) on the server, allowing users to interact with Code Puppy from any web browser. The interface includes connection status indicators, session management, and terminal controls.

Edge Cases & Gotchas

WARNING

WebSocket reconnection: If the WebSocket disconnects, the user must manually click "Reconnect". Automatic reconnection is limited to 5 attempts.

NOTE

Session isolation: Each session is isolated. Multiple browser tabs can run independent Code Puppy instances.

TIP

Terminal resize: The terminal automatically adjusts to window size. PTY is notified of resize events.

CAUTION

Local access only: The web interface is intended for local development. CORS and security should be configured for production use.

Security Considerations

CAUTION

No authentication: The web interface currently has no authentication. Only use on trusted networks.

CAUTION

PTY access: Anyone with access to the terminal can execute arbitrary commands. Restrict access appropriately.
