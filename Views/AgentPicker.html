View: Agent Picker

Overview

Property Value
Route /agent (slash command)
Auth Required N/A (local CLI)
Layout Split-panel (35% / 65%)

What Does This View Do?

The Agent Picker is an interactive terminal UI that allows users to browse, select, and manage AI agents. It provides a split-panel interface showing the agent list on the left and detailed preview information on the right. Users can switch agents, pin models, clone agents, and delete clones.

Agent Data Flow

sequenceDiagram
    participant User
    participant Picker
    participant AgentMgr
    participant Config
    
    User->>Picker: Open agent picker
    Picker->>AgentMgr: Get available agents
    AgentMgr-->>Picker: Agent list
    Picker->>User: Display list + preview
    User->>Picker: Select agent
    alt Pin model
        Picker->>User: Show model picker
        User->>Picker: Select model
        Picker->>Config: Save pinned model
        Picker->>AgentMgr: Reload agent
    else Clone agent
        Picker->>AgentMgr: Clone agent
        AgentMgr-->>Picker: New agent name
        Picker->>User: Refresh list
    else Delete clone
        User->>Picker: Confirm delete
        Picker->>AgentMgr: Delete agent
        AgentMgr-->>Picker: Success
        Picker->>User: Refresh list
    else Switch agent
        Picker->>AgentMgr: Switch to agent
        AgentMgr-->>Picker: Success
        Picker-->>User: Return agent name
    end

Edge Cases & Gotchas

WARNING

Cannot delete built-in agents: Only cloned agents can be deleted. Attempting to delete a built-in agent shows a warning.

WARNING

Cannot delete active agent: The currently active agent cannot be deleted. Must switch to a different agent first.

NOTE

Pagination: Agents are paginated with 10 per page. Use ← → to navigate pages.

TIP

Model pinning: Pinned models persist across sessions. Use P to pin a specific model to an agent.

CAUTION

Terminal buffer: The picker uses an alternate screen buffer. If interrupted, the terminal may need resetting.

Text Sanitization

The picker sanitizes all display text to avoid terminal rendering issues:

  • Emojis and special symbols are stripped
  • Wide characters are removed
  • Only safe Unicode categories are kept (letters, numbers, basic punctuation)

This prevents display corruption in various terminal emulators.
