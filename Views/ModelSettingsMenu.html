View: Model Settings Menu

Overview

Property Value
Route /settings <model-name> (slash command)
Auth Required N/A (local CLI)
Layout Split-panel (models/settings)

What Does This View Do?

The Model Settings Menu provides an interactive terminal UI for configuring per-model AI parameters. Users can select a model and adjust settings like temperature, seed, reasoning effort, and verbosity. Settings are stored per-model and persist across sessions.

Edge Cases & Gotchas

WARNING

Model support: Not all settings are available for all models. Unsupported settings are hidden.

NOTE

Global vs per-model: Some settings (reasoning_effort, verbosity) have global defaults that apply if no per-model setting exists.

TIP

Reset to default: Set a setting to None to use the model's default value.

CAUTION

Invalid values: Values outside the valid range are clamped to min/max.
