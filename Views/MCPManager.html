View: MCP Manager

Overview

Property Value
Route /mcp (slash command)
Auth Required N/A (local CLI)
Layout Split-panel (categories/servers)

What Does This View Do?

The MCP Manager provides an interactive terminal UI for managing Model Context Protocol (MCP) servers. Users can browse available servers from a catalog, install new servers, start/stop/restart servers, view server status, and configure custom MCP servers.

Edge Cases & Gotchas

WARNING

Server conflicts: Installing a server with the same name as an existing one will fail. Remove the existing server first.

NOTE

Custom servers: Custom servers can be added via the "➕ Custom Server" category with manual configuration.

TIP

Server restart: After configuration changes, use /mcp restart <name> to apply changes.

CAUTION

Server dependencies: Some servers require external tools (e.g., npx, docker). Ensure dependencies are installed.
