View: Onboarding Wizard

Overview

Property Value
Route First run / /tutorial command
Auth Required N/A (local CLI)
Layout Full-screen single panel

What Does This View Do?

The Onboarding Wizard is a 5-slide interactive tutorial that guides first-time users through Code Puppy setup. It helps users select their preferred AI model, understand available features, and optionally configure OAuth authentication. The wizard is ADHD-friendly with clear visual progress indicators and simple navigation.

User Actions

Action Trigger Result
Next slide → / Enter / l Advance to next slide
Previous slide ← / h Go to previous slide
Select option ↓ / j / Ctrl+N Move selection down
Select option ↑ / k / Ctrl+P Move selection up
Skip wizard ESC Exit without setup
Cancel Ctrl+C Exit without setup

Slide Sequence

flowchart LR
    A[Welcome] --> B[Model Selection]
    B --> C[MCP Servers]
    C --> D[Use Cases]
    D --> E[Done]

Slide 1: Welcome

  • Introduction to Code Puppy
  • Key features overview
  • Navigation hints

Slide 2: Model Selection

  • Choose default AI model provider
  • Options: ChatGPT, Claude, or skip
  • Triggers OAuth setup if selected

Slide 3: MCP Servers

  • Explanation of Model Context Protocol
  • Benefits of MCP servers
  • Can be configured later

Slide 4: Use Cases

  • Planning vs Coding modes
  • When to use each
  • Best practices

Slide 5: Done!

  • Summary of choices
  • Next steps
  • OAuth setup link (if applicable)

Edge Cases & Gotchas

WARNING

One-time only: The wizard only shows on first run. Use /tutorial to re-run.

NOTE

Skip option: Users can press ESC at any time to skip the wizard.

TIP

Reset onboarding: Delete ~/.code_puppy/onboarding_complete to re-run the wizard.

CAUTION

OAuth triggers: Selecting ChatGPT or Claude triggers OAuth flow immediately.
