Endpoints Index

Overview

REST API endpoints for programmatic access to Code Puppy functionality. All endpoints are local-only (no authentication required).

Base URL

http://localhost:8000

Agents API

List Agents

GET /api/agents - Get all available agents

Commands API

List Commands

GET /api/commands - Get all available commands

Get Command

GET /api/commands/{name} - Get command details

Execute Command

POST /api/commands/execute - Execute a command

Command Autocomplete

POST /api/commands/autocomplete - Get command completions

Config API

Get Config

GET /api/config - Get all configuration

List Config Keys

GET /api/config/keys - List all config keys

Get Config Key

GET /api/config/{key} - Get a specific config value

Set Config Key

PUT /api/config/{key} - Set a config value

Delete Config Key

DELETE /api/config/{key} - Delete a config value

Sessions API

List Sessions

GET /api/sessions - List all sessions

Get Session

GET /api/sessions/{id} - Get session details

Get Session Messages

GET /api/sessions/{id}/messages - Get session messages

Delete Session

DELETE /api/sessions/{id} - Delete a session
