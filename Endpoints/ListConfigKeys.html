Endpoint: GET /api/config/keys

Overview

Property Value
Method GET
Path /api/config/keys
Auth public
Rate Limit N/A

What Does This Do?

Returns a list of all valid configuration keys that can be used with the config API. This includes both keys currently set in puppy.cfg and all known/default configuration keys that Code Puppy recognizes.

Use this when you need to:

  • Discover available configuration options
  • Validate config keys before setting them
  • Build dynamic configuration UIs
  • Integrate with external configuration tools

Key Categories

Core Settings

  • model - Currently selected model
  • default_agent - Default agent to use
  • temperature - Model temperature
  • yolo_mode - Skip safety confirmations

Message History

  • message_limit - Max messages to keep
  • compaction_strategy - How to compact history
  • protected_token_count - Tokens to protect from compaction
  • compaction_threshold - Trigger compaction at this token count

Agent Behavior

  • allow_recursion - Allow recursive agent calls
  • enable_universal_constructor - Enable dynamic tool creation
  • enable_pack_agents - Enable specialized sub-agents
  • enable_streaming - Enable streaming responses
  • subagent_verbose - Verbose sub-agent output

Session Management

  • auto_save_session - Auto-save on exit
  • max_saved_sessions - Max sessions to retain
  • resume_message_count - Messages to show on resume

UI/UX

  • puppy_name - Custom name for the agent
  • owner_name - Custom owner name
  • diff_context_lines - Diff context lines
  • cancel_agent_key - Key to cancel agent

Frontend Integration

  • frontend_emitter_enabled - Enable frontend event emitter
  • frontend_emitter_max_recent_events - Max recent events to keep
  • frontend_emitter_queue_size - Event queue size

System

  • http2 - Use HTTP/2
  • enable_dbos - Enable DBOS persistence

Banner Colors

  • banner_color_{name} - Color for named banners (e.g., banner_color_default)
