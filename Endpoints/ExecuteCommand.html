Endpoint: POST /api/commands/execute

Overview

Property Value
Method POST
Path /api/commands/execute
Auth public
Rate Limit N/A

What Does This Do?

Executes a slash command (e.g., /set model=gpt-4o) via the REST API. This allows external tools and web interfaces to trigger Code Puppy's built-in commands without using the terminal UI.

Request

Headers

Header Required Description
Content-Type yes application/json

Path Parameters

None

Query Parameters

None

Request Body

{
  "command": "string - Full command string with or without leading /"
}

Example Request Body

{
  "command": "/set model=gpt-4o"
}

Validation Rules

Field Rules
command Required, non-empty string. Leading / is optional.

Response

Success Response (200)

{
  "success": true,
  "result": "Command output or return value",
  "error": null
}

Error Response (200 with failure)

{
  "success": false,
  "result": null,
  "error": "Error message describing what went wrong"
}

Error Scenarios

Status When
200 (success=false) Command execution failed
200 (timeout) Command timed out after 30s
400 Invalid request body

Flow

sequenceDiagram
    participant Client
    participant Router
    participant Handler
    participant Registry

    Client->>Router: POST /api/commands/execute
    Router->>Router: Validate request body
    Router->>Router: Strip leading / if present

    end
    Handler-->>Router: Result
    Router->>Router: Wrap in CommandExecuteResponse
    Router-->>Client: JSON response

Examples

Example Request

curl -X POST 'http://localhost:8000/api/commands/execute' \
  -H 'Content-Type: application/json' \
  -d '{"command": "/set model=gpt-4o"}'

Example Success Response

{
  "success": true,
  "result": true,
  "error": null
}

Example Command Not Found Response

{
  "success": false,
  "result": null,
  "error": "Unknown command: /unknown_cmd"
}

Example Timeout Response

{
  "success": false,
  "result": null,
  "error": "Command timed out after 30.0s"
}

Edge Cases & Gotchas

WARNING

30-second timeout: Long-running commands will be terminated. Use this endpoint for quick configuration commands, not for triggering agent runs.

NOTE

Leading slash is optional: Both /set model=gpt-4o and set model=gpt-4o work identically.

NOTE

Plugin commands: Unknown commands are passed to plugins via the custom_command callback before returning "unknown command" warning.
