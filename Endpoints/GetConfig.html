Endpoint: GET /api/config

Overview

Property Value
Method GET
Path /api/config
Auth public
Rate Limit N/A

What Does This Do?

Returns all configuration keys and their current values in a single response. This endpoint provides a complete snapshot of the Code Puppy configuration as it exists in puppy.cfg, merged with all known/default configuration keys.

Use this when you need to:

  • Inspect the full configuration state
  • Build configuration UI dashboards
  • Debug configuration-related issues
  • Export configuration for backup/transfer

Implementation Map

Layer Location Notes
Route definition code_puppy/api/routers/config.py Line 18-24
Handler code_puppy/api/routers/config.py:list_config() Iterates all keys
Config reader code_puppy/config.py:get_config_keys() Returns merged key list
Value getter code_puppy/config.py:get_value() Reads from puppy.cfg

Flow

sequenceDiagram
    participant Client
    participant Router
    participant ConfigAPI
    participant ConfigModule
    
    Client->>Router: GET /api/config
    Router->>ConfigAPI: list_config()
    ConfigAPI->>ConfigModule: get_config_keys()
    ConfigModule-->>ConfigAPI: List of all keys
    loop For each key
        ConfigAPI->>ConfigModule: get_value(key)
        ConfigModule-->>ConfigAPI: Config value
    end
    ConfigAPI-->>Router: {"config": {...}}
    Router-->>Client: JSON response

Configuration Keys

Core Settings

Key Type Default Description
model string gpt-5 Currently selected model
default_agent string code-puppy Default agent to use
temperature float 0.7 Model temperature
yolo_mode boolean false Skip safety confirmations

Message History

Key Type Default Description
message_limit int 100 Max messages to keep
compaction_strategy string protect_summarize How to compact history
protected_token_count int 50000 Tokens to protect from compaction
compaction_threshold int 100000 Trigger compaction at this token count

Agent Behavior

Key Type Default Description
allow_recursion boolean true Allow recursive agent calls
enable_universal_constructor boolean true Enable dynamic tool creation
enable_pack_agents boolean false Enable specialized sub-agents
enable_streaming boolean true Enable streaming responses
subagent_verbose boolean false Verbose sub-agent output

Session Management

Key Type Default Description
auto_save_session boolean false Auto-save on exit
max_saved_sessions int 10 Max sessions to retain
resume_message_count int 50 Messages to show on resume

UI/UX

Key Type Default Description
puppy_name string Puppy Custom name for the agent
owner_name string Master Custom owner name
diff_context_lines int 3 Diff context lines
cancel_agent_key string ctrl+c Key to cancel agent

Frontend Integration

Key Type Default Description
frontend_emitter_enabled boolean false Enable frontend event emitter
frontend_emitter_max_recent_events int 100 Max recent events to keep
frontend_emitter_queue_size int 1000 Event queue size

System

Key Type Default Description
http2 boolean true Use HTTP/2
enable_dbos boolean true Enable DBOS persistence

Banner Colors

Key Pattern Type Default Description
banner_color_{name} string varies Color for named banners
