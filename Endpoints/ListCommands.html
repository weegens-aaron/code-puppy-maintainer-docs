Endpoint: GET /api/commands

Overview

Property Value
Method GET
Path /api/commands
Auth public
Rate Limit N/A

What Does This Do?

Returns a sorted list of all available slash commands with their metadata, including name, description, usage, aliases, category, and detailed help text. This endpoint is used by external tools and UIs to discover what commands are available in Code Puppy.

Request

Headers

Header Required Description
No specific headers required no

Path Parameters

None

Query Parameters

None

Request Body

None

Response

Success Response (200)

Returns a JSON array of CommandInfo objects, sorted alphabetically by command name.

CommandInfo Schema

Field Type Description
name string The primary command name (without leading /)
description string Short description of what the command does
usage string Usage syntax showing required/optional arguments
aliases string[] Alternative names for the command
category string Command category (e.g., "core", "models", "agents", "mcp")
detailed_help string \ null Extended help text, or null if not available

Error Responses

Status When
500 Internal server error (e.g., command registry unavailable)

Edge Cases & Gotchas

NOTE

No duplicates: The endpoint returns unique commands only - aliases are included in each command's aliases array, not as separate entries.

NOTE

Sorted output: Commands are always returned in alphabetical order by name, making it predictable for consumers.

NOTE

Case-insensitive aliases: When looking up commands by alias, the lookup is case-insensitive (handled by the get_command() function used by other endpoints).
