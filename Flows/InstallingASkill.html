Flow: Installing a Skill

What Happens?

This flow handles the complete journey of browsing, selecting, downloading, and installing a remote agent skill from the skill catalog into the local ~/.code_puppy/skills/ directory. It includes safety validations, user confirmation, and post-install verification.

Trigger

  • User action: /skills install command in the CLI
  • UI: Interactive split-panel TUI for browsing categories and skills

Outcome

  • Skill ZIP downloaded from remote URL
  • Extracted and validated for safety (zip-slip protection)
  • Installed to ~/.code_puppy/skills/<skill_id>/
  • Skill cache refreshed for immediate availability
  • User sees success/error feedback

1. Launch TUI

  • What: Opens split-panel prompt_toolkit interface
  • Where: code_puppy/plugins/agent_skills/skills_install_menu.py
  • Failure mode: TUI fails to start, returns False immediately

2. Load Remote Catalog

  • What: Fetches skill catalog from remote source with cache fallback
  • Where: code_puppy/plugins/agent_skills/remote_catalog.py → SkillCatalog.__init__()
  • Failure mode: Catalog empty, shows "No remote categories available" message

3. Browse Categories

  • What: Displays paginated list of skill categories with skill counts
  • Where: SkillsInstallMenu._render_category_list()
  • Navigation: Arrow keys to navigate, Enter to enter category, Esc to exit

4. Browse Skills

  • What: Shows skills in selected category with install status indicators
  • Where: SkillsInstallMenu._render_skill_list()
  • Status icons: ✓ = installed, ○ = not installed

5. View Details

  • What: Displays skill metadata: description, size, download URL, contents
  • Where: SkillsInstallMenu._render_details()
  • Info shown: ID, description, category, tags, file count, ZIP size

6. Confirm Installation

  • What: Prompts user with confirmation before downloading
  • Where: _prompt_and_install()
  • Message format: Install skill '<name>' (<size>)? [y/N]
  • Cancellation: Any input except y/yes cancels

7. Download ZIP

  • What: Streams ZIP file from remote URL to temp directory
  • Where: download_and_install_skill() → _download_to_file()
  • Security: 30s timeout, streaming download, User-Agent header
  • Failure mode: Network error, HTTP error, returns InstallResult(success=False)

8. Validate ZIP Safety

  • What: Checks for zip-slip path traversal and size limits
  • Where: _validate_zip_safety()
  • Checks:
  • Absolute paths (starting with /)
  • Parent traversal (.. in path)
  • Total uncompressed size > 50MB
  • Failure mode: Unsafe ZIP rejected, error shown to user

9. Extract to Temp

  • What: Safely extracts ZIP contents to temporary directory
  • Where: _safe_extract_zip()
  • Security: Each entry validated to stay within extract directory
  • Skips: __MACOSX metadata folders
  • Failure mode: Extraction error, partial cleanup, error shown

10. Verify SKILL.md

  • What: Validates extracted layout contains required SKILL.md
  • Where: _determine_extracted_root()
  • Layouts supported:
  • Files at ZIP root with SKILL.md
  • Single top-level folder containing SKILL.md
  • Failure mode: Missing SKILL.md, installation aborted

11. Move to Skills Directory

  • What: Copies extracted content to ~/.code_puppy/skills/<skill_id>/
  • Where: _stage_normalized_install() → shutil.move()
  • Overwrite: Existing install deleted if force=True
  • Failure mode: Move error, partial install cleaned up

12. Refresh Cache

  • What: Updates local skill cache so new skill is immediately available
  • Where: refresh_skill_cache()
  • Note: Cache refresh failure doesn't poison successful install
  • Failure mode: Logged as warning, install still succeeds

Data Transformations

Stage Input Output Location
Catalog fetch Remote JSON SkillCatalogEntry[] remote_catalog.py
Category browse Category name SkillCatalogEntry[] SkillCatalog.get_by_category()
Download URL Local ZIP file _download_to_file()
Extract ZIP file Extracted directory _safe_extract_zip()
Normalize Extracted root Staged skill dir _stage_normalized_install()
Install Staged dir ~/.code_puppy/skills/<id>/ shutil.move()

Failure Handling

Failure Point Detection Recovery Alert?
Catalog unavailable Exception during fetch Show empty menu Info message
Download failure HTTP error, timeout Return InstallResult Error message
Unsafe ZIP Path traversal detected Reject, don't extract Warning + error
Invalid ZIP BadZipFile exception Return error Error message
Missing SKILL.md File not found after extract Abort, cleanup Error message
Move failure shutil.move() exception Cleanup partial install Error message
Cache refresh Exception Log warning, continue Warning only

Performance Characteristics

  • Typical duration: 5-30 seconds (depends on ZIP size and network)
  • Bottlenecks: Network download speed, ZIP extraction (single-threaded)
  • Memory: ZIP size + extraction size (capped at 50MB uncompressed)
  • Disk: Temporary storage in system temp, then final location

Debugging This Flow

Key Log Messages

Message Meaning Action
Failed to fetch remote catalog: {e} Network/cache issue Check network, retry
Rejected unsafe zip for {name}: {reason} Security block Verify ZIP source
Downloaded file is not a valid zip Corrupted download Retry download
Extracted zip missing SKILL.md Invalid skill package Contact skill author
Installed skill '{name}' into {path} Success Verify with /skills list

Common Issues

Symptom Likely Cause Fix
"No remote categories available" Network down, cache empty Check network, wait for cache
Download hangs Slow/unreachable server Cancel, retry later
"Unsafe zip entry" Malicious or malformed ZIP Don't install, report issue
"Failed to extract" Corrupted ZIP Retry download
Skill not appearing after install Cache not refreshed Run /skills refresh manually

Manual Verification

# Check if skill is installed
ls -la ~/.code_puppy/skills/<skill_id>/

# Verify SKILL.md exists
cat ~/.code_puppy/skills/<skill_id>/SKILL.md

# Refresh cache manually
code_puppy /skills refresh
