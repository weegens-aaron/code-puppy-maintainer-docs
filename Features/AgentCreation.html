Feature: Agent Creation

What Does This Do?

Agent Creation is the interactive system that helps users design, validate, and create new JSON agent configuration files. It guides users through defining an agent's purpose, selecting appropriate tools, optionally pinning a specific model, and generates a properly structured JSON file ready to use.

Why Does This Exist?

Code Puppy supports both Python-based agents (hardcoded classes) and JSON-based agents (configurable files). The Agent Creator makes it easy for users to:

  • Create custom agents without writing Python code
  • Experiment with different agent configurations
  • Share and version control agent definitions
  • Learn about available tools and best practices

How It Works

User Perspective

  1. User invokes the Agent Creator via /agent menu or direct command
  2. Agent Creator asks what the new agent should do
  3. Based on the purpose, it suggests appropriate tools
  4. User confirms tool selection and optionally pins a model
  5. Agent Creator generates a complete JSON configuration
  6. User reviews and confirms
  7. Agent Creator automatically creates the JSON file
  8. User can immediately use the new agent with /agent <name>

Edge Cases & Gotchas

WARNING

Agent Creator auto-creates files after confirmation - Once you confirm the generated JSON, the file is created immediately. There's no second confirmation step.

NOTE

Model pinning is optional - If you don't pin a model, the agent uses the global default. You can pin later with /pin_model <agent-name> <model-name>.

TIP

Tool selection matters - Only include tools the agent actually needs. Too many tools can confuse the agent; too few can limit its capabilities.

Error Scenarios

Scenario Behavior User Sees
Invalid tool name Validation fails "Invalid tools: [tool-name]. Available: [...]"
Missing required field Validation fails "Missing required field: 'name'"
Agent already exists Creation blocked "Agent 'name' already exists at path"
Spaces in name Validation fails "'name' should not contain spaces (use hyphens)"

Testing This Feature

  1. Run Code Puppy: uv run code_puppy
  2. Open agent menu: /agent
  3. Select "Create a new agent"
  4. Follow the interactive prompts
  5. Verify the JSON file is created in ~/.code_puppy/agents/
  6. Test the new agent by invoking it
